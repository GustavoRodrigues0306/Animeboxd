# AnimeBoxd

AnimeBoxd is a web application inspired by Letterboxd, designed specifically for anime enthusiasts. The platform allows users to discover anime, view detailed information, manage personal watchlists, save favorites, track watched series, and write reviews.

## Project Overview

The goal of AnimeBoxd is to practice modern front-end development using pure HTML, CSS, and JavaScript while building a complete and interactive user experience.

The project focuses on API consumption, DOM manipulation, local data persistence, and interface design.

---

## Features

### Anime Discovery

* Search anime by title
* Browse popular anime
* View detailed information about each anime

### User Library

* Save anime to favorites
* Mark anime as watched
* Manage personal anime collections

### Reviews and Ratings

* Rate anime titles
* Write custom reviews and comments
* Store user interactions locally

### User Experience

* Light and dark theme support
* Toast notifications for user feedback
* Responsive interface for different screen sizes

## API

This project uses the Jikan API to retrieve anime information from MyAnimeList.

API Endpoint:

https://api.jikan.moe/v4

Available data includes:

* Title
* Synopsis
* Rating
* Episodes
* Images
* Genres
* Release Information

---

## Local Storage

The application uses LocalStorage to simulate persistent user data, including:

* Registered users
* Logged-in user
* Favorite anime
* Watched anime
* Reviews and ratings
* Selected theme preferences
